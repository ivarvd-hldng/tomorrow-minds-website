# Tomorrow Minds Website

Official website for Tomorrow Minds AI consultancy, built with React.

## Development

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build
```

## Blog
Blog posts are written in Markdown and stored in `content/blog/`. Each post should include frontmatter with title, date, and excerpt.

## Contact Form
The contact form is handled by Netlify Forms. Form submissions will be available in the Netlify dashboard.
